Isolated Working Game Example

Ridge Command — 2D RTS micro prototype

Standalone HTML game loop with selectable units, smart auto-engagement, ghosted build placement, timed production queues, escalating waves with multiple enemy types, particle hits, and a real victory state. No screenshots, no frameworks — just a single file you can open in a browser.

  • LMB click or drag to select. Shift+click to add to selection.
  • RMB on ground to move, on enemy to attack. A then click for attack-move.
  • Q Worker · W Soldier · E Depot · R Turret · Space Pause.
  • S Stop · H Hold · Ctrl+1-3 bind group · 1-3 recall.
  • Cliffs block sight and shots. Hold high ground for +range and +damage.
