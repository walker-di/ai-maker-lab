## Development-sync metrics

Because the main bottleneck is sync between development and recording, track operational metrics too.

| Metric | Definition | Goal |
|---|---|---|
| Dev content notes captured | Number of dev sessions with a content note | Increase consistency |
| Notes converted to shorts | Percent of notes turned into shorts | Identify easy wins |
| Notes converted to weekly segments | Percent used in weekly videos | Keep videos synced |
| Recording lag | Days between development event and recording | Reduce |
| Topic backlog age | Average age of unrecorded high-score topics | Reduce |
| Deep-dive graduation rate | Weekly topics that become evergreen videos | Use metrics to choose |

## Decision rules

| Signal | Interpretation | Action |
|---|---|---|
| Low CTR | Title/thumbnail/topic packaging is weak | Rewrite title, make result clearer |
| Low 30s retention | Hook is too slow | Start with conflict/result |
| Low avg view % | Video unfocused or too long | Narrow to one idea, add demos |
| High comments but low views | Valuable topic, weak distribution | Clip it, retitle, repurpose |
| High search traffic | Evergreen demand | Create deeper follow-up |
| High production effort + weak metrics | Format is too expensive | Downgrade to build-log segment |

## Title and thumbnail testing rules

### Title patterns to test

```text
I Built [specific system] to Solve [specific problem]
How I Use [AI workflow] to [result]
The Missing Piece in [AI coding/product workflow]
Why [common approach] Breaks When [real constraint]
Building [project]: [specific episode result]
```

Examples:

- `My AI Development Is Faster Than My Content Process`
- `I’m Building an AI Software Team: CEO, Orchestrator, Coder, QA, Designer`
- `The Missing Piece in AI Coding: Real QA Evidence`
- `I Tried Turning My Development Process Into a Content Calendar`

### Thumbnail principles

- Show one clear contrast, not many labels.
- Prefer visual artifacts: agent org chart, QA screenshot, sprint board, architecture diagram.
- Use short text: 2–5 words.
- Avoid vague labels like `Update` or `AI Stuff`.
